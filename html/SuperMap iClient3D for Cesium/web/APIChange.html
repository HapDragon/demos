产品变更信息

当前版本：11i(2022)Beta
上次版本：10i(2021)SP1

1. 接口变更：

类名称

新增接口

接口说明

ProjectionImage

 setFusionLayers(layers) 设置视图融合
layers：设置图片融合的S3M图层，类型：Array

Globe

 granularity 设置地形开挖精度，越小越精确，类型：Number，默认值：1
removeAllExtractRegion() 移除所有开挖区域和抽出部分
removeExtractRegion() 移除当前开挖区域和抽出部分
addExtractRegion(options) 地形开挖抽出显示
name：该开挖区域的名称，类型 : String
position：地形开挖区域，类型：Array
height：地形开挖深度，类型：Number
transparent：封边是否透明，类型：Boolean，默认值：false
extractHeight：地形开挖抽出高度，类型：Number，默认值：100
addImageryClipRegions(options) 设置影像裁剪
region：裁剪区域的经纬度点集，类型：Array
layers：参与裁剪的影像图层，类型：Array
name：该裁剪区域的name（key值），类型 : String
removeImageryClipRegion(name,layers) 移除裁剪区域
layers：参与裁剪的影像图层，类型：Array
name：该裁剪区域的name（key值），类型 : String

S3MTilesLayer

 setBatchObjsTranslate(option) 批量设置对象偏移
option：拟移除偏移的对象的id和偏移量集合，类型：Object
heightScale 设置模型z方向拉升高度值，类型：Number，默认值：1

FloodAnalysis3D

 WaterMode 设置连通性淹没分析模式，类型：WaterMode
